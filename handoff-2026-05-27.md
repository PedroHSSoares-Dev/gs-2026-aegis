## 1. Objetivo

Construir o **AEGIS Mission Control**, um dashboard de monitoramento de desastres planetários para o Global Solution 2026 da FIAP (grupo Predictfy). A interface simula uma central de comando com mapa mundial interativo, feed de eventos climáticos, timeline temporal e painéis analíticos. O projeto é mock-first (dados estáticos), estruturado para receber dados reais da NASA API na Fase 2.

- Nenhum comentário explicando "o quê" — só o "porquê" quando não-óbvio

**Decisões tomadas:**
- Severity (critical/high/medium/low) como sistema de cores único — `kindColor` foi removido; o shape do ícone identifica o tipo, a cor identifica a urgência
- Heat overlays usam gradientes radiais por severity (não por kind como no protótipo HTML original)
- AEF **nunca** filtra por scrub da timeline — scrub só afeta opacidade dos eventos no mapa
- `expanded` state no App.tsx controla o card aberto no AEF; `selected` controla zoom no mapa; os dois andam juntos via `handleSelect`
- Zoom no mapa é **programático** (zoom-to-event ao selecionar), não por gestos do usuário — os handlers de wheel/drag foram removidos intencionalmente
- `scrub=0.5` = NOW, `scrub=0` = D+7 (futuro), `scrub=1` = D-7 (passado) — esse eixo foi invertido em relação ao protótipo original e está documentado em `Timeline.tsx`
- `.has-selection` class adicionada via JS (não CSS `:has()`) porque EventCards ficam dentro de `EventGroup > event-list`, quebrando o seletor `:has()` original

**Snippets críticos:**

`scrubToDaysOffset` (Timeline.tsx — exportado e usado em App.tsx):
```ts
export function scrubToDaysOffset(value: number): number {
  return (value - 0.5) * 14; // 0→-7, 0.5→0, 1→+7
}
```

`handleSelect` (App.tsx — toggle zoom+expand):
```ts
function handleSelect(id: string) {
  if (selected === id) {
    setSelected('');
    setExpanded(null);
  } else {
    setSelected(id);
    setExpanded(id);
  }
}
```

`isReplay` (App.tsx):
```ts
const isReplay = Math.abs(scrub - 0.5) > 0.03;
```

`has-selection` dimming (App.tsx JSX):
```tsx
<div className={`event-list${expanded ? ' has-selection' : ''}`}>
```

Tipos principais (`src/types/index.ts`):

export type EventKind = 'wildfire' | 'cyclone' | 'flood' | 'quake' | 'drought';
export type Severity  = 'critical' | 'high' | 'medium' | 'low';

export interface DisasterEvent {
